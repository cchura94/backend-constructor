### Paquetes
- Instalar los siguientes paquetes:
```
npm install express
npm install dotenv
```
- Para realizar peticiones al servidor de Meta (WhaTSAPP) Instalamos:
```
npm install axios
```
### Paquetes para Bases de datos
- Instalar Sequelize
````
npm install --save sequelize
npm install --save pg pg-hstore
```
